Project-Pier by dbernar1

GitHub repo for the Project Pier project management OS software

clone the git repository ∘ git clone https://github.com/dbernar1/Project-Pier.git • install the system ∘ usual installation steps, see them here: http://www.projectpier.org/manual/installation • set up git ignores for the files that are installation specific ∘ git update-index --assume-unchanged cache/autoloader.php ∘ git update-index --assume-unchanged cache/log.php ∘ git update-index --assume-unchanged config/empty.config.php ∘ git update-index --assume-unchanged config/config.php ∘ caveat about ever wanting to commit changes to the files that are ignored
